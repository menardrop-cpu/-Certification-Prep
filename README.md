# -Certification-Prep
Study notes for my cybersecurity certification path.
## Target certifications

| Certification | Status | Target date |
|---------------|--------|-------------|
| ISC2 CC | In progress | 2026 |
| ISO 27001 Lead Implementer | Planned | 2027 |

## What is in here

Structured notes by domain, condensed summaries of key concepts, 
and personal flashcards. Written to retain, not to impress.

No bloated PDFs. Just the essentials, in my own words.

## ISC2 CC domains covered

* Security principles
* Business continuity, disaster recovery and incident response
* Access controls concepts
* Network security
* Security operations

## Study approach

Read the official material, rewrite it in plain language, 
test with practice questions, document what I got wrong and why.
